🎸 Drop C + 指弾きのトリック

  • Drop C (C-A-D-G-B-E) でローCを開放弦に
  • 親指(Thumb)が6弦を常に担当 — コード+ベースライン同時演奏
  • コードは「implied」— ギターとベースの合算で成立
  • Swing 16thsで生まれる独特のグルーヴ感
  • B♭§ の § は特殊チューニング専用フォーム

🎵 コード機能 解析

  • Csus4 — IIsus4。解決しない永続的緊張
  • A♭sus2 / A♭§ — ♭VII。B♭ミクソリディアン
  • G♭§ — ♭VIb。プレコーラスの半音下降色
  • Bridge上昇 — Fm→Gm→A♭m→B♭m（全音+半音）
  • C7♯9 — II7♯9。ブルース的インタールード
  • G/B — ブリッジ末の予想外解放コード

📐 使用スケール

  • B♭ Major — 基本スケール
  • B♭ Mixolydian — A♭, G♭ 由来
  • B♭ Minor (借用) — Bridge の B♭m7add4
  • Blues scale (C) — Interlude C7♯9
  • ペンタ + テンションの融合 = John Mayer style

🌃 "Neon" とハーモニー

夜のアトランタ・ピーチツリー通りを流れるネオンの光の如く、Csus4が解決せずに漂い続けるのがこの曲の核心。

G♭§という半音下降が「色が変わる瞬間」を、C7♯9が「夜の深みと混乱」を表し、Bridge の上昇が「光が上へ伸びる瞬間」を音楽化している。

Outro は同じコードでフェードアウト。ネオンは消えても通りは残る。
